# WantedCarSearch
Для кращого розуміння практичної площини застосування подібних алгоритмів реалізувати наступну програму.
Нехай у нас є база даних автомобілів  що знаходяться в розшуку задана номерним знаком (для простоти алгоритму задане числом) та власником авто. Автоматична камера фіксації на автошляху фіксує номерний знак і в разі фіксації авто що знаходиться в розшуку повідомляє систему вказуючи адресу камери номер авто та назву власника авто. Симулювати алгоритм  в консолі де роль автоматичної камери фіксації бе виконувати ввід номера з консолі користувача, а нотифікація системи буде відбуватись у вигляді логування в консоль.
